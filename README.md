# Guess My Number Game

#### Guess My Number Game, 10/03/2021

#### By **Wilson Mwangeka**

## Description

Guess my number game is a game where you have to guess a randomly generated by the computer correctly before your score which starts at 20 gets to zero. The faster you can accurately guess this number the higher your highscore.

## Setup/Installation Requirements

* Load the page at (https://williz4.github.io/guess-my-number/). 

## Known Bugs
There are currently no known bugs in this website.

## Technologies Used

I used HTML5 and CSS3 for the structure and design respectively and Javascript to be able to do the dom manipulation and to be able to implement the game logic.
